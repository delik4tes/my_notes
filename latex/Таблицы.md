- **\begin{table}** - Тег начинает таблицу
- **\caption**{Название таблицы **\cite{L1**}} - Название таблицы и ссылка на источник
- **\label{tab:5-4**} - Метка, на которую потом можно будет ссылаться
- **\begin{tabular}{|c|c|c|}** - Таблица начнётся после этого тега; три столбца, выравненные по центру 
- **\hline** - Добавляет горизонтальную лини
- **\begin{tabular}{|p{2cm}|p{2cm}|p{2cm}|p{2cm}|p{2cm}|p{2cm}|}** - Выравнивае столбцов по ширине. Теперь слишком широкий текст автоматически будет переноситься на следующую строку внутри столбца.
> Чтобы объединить столбцы можно воспользоваться тегом **\multicolumn{2}{|c|}- где {2}** - это число столбцов, которые нужно объединить **{|c|}** - выравнивание по центру
